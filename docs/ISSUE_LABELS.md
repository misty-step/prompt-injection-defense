# Issue Labels

## Kind
- `kind:experiment`: hypothesis-driven research runs and experiment planning.
- `kind:bug`: defects in harnesses, scoring, analysis, docs, or infra.
- `kind:infra`: repo/tooling/process improvements.

## Area
- `area:prompt-injection`: prompt injection experiment program.
- `area:tooling`: developer tooling, scripts, automation.
- `area:data`: schemas, artifacts, reproducibility.

## Priority
- `priority:p0`: blocks valid experiment execution or results trust.
- `priority:p1`: important, should ship this cycle.
- `priority:p2`: backlog, useful but not urgent.
